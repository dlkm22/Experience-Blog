It is Week 2 of the assignment and progress has been made!

We have come up with a more elaborate and detailed structure for our idea. Below is a picture of what it will look like.

As seen above, we have decided to use an RFID reader instead of an NFC reader. This is because the RFID reader we're looking at has a further detection range, making it perfect for our 'gantry'-like system.

We plan to use a Raspberry Pi to handle the decoding of an employee's RFID card. Raspberry Pi seen below.

We will use Python and MSSQL to handle the receiving and inserting of data. Our basic goal for now is to get the Python and MSSQL database connected. Through my research, it seems that we need a Python Driver to communicate with MSSQL.

This is a link for the Python Driver. Here

We have also found an already written Python Script that reads an RFID card from a serial RFID reader attached to a Raspberry Pi.

Many advancements have been made in our 2nd Week, hopefully we can finish a prototype by Week 5 for the hackathon.
